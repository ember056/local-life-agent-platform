# 接口文档

本文档只列出项目中的核心接口，方便测试和面试展示。

## 用户接口

### 发送验证码

```http
POST /user/code
```

参数：

- `phone`：手机号

### 登录

```http
POST /user/login
```

请求体：

```json
{
  "phone": "13800000000",
  "code": "123456"
}
```

返回 token。后续请求需要在请求头中携带 `authorization`。

## 商户接口

### 查询商户详情

```http
GET /shop/{id}
```

说明：优先查 Redis，未命中时查 MySQL 并写回缓存。

### 搜索商户

```http
GET /shop/search?keyword=火锅&page=1&size=10
```

说明：走 Elasticsearch 商户索引。

### 同步商户到 ES

```http
POST /shop/es/sync
```

说明：把 MySQL 中的商户数据同步到 Elasticsearch。

## 优惠券接口

### 查询店铺优惠券

```http
GET /voucher/list/{shopId}
```

### 秒杀优惠券

```http
POST /voucher-order/seckill/{id}
```

说明：

1. Redis Lua 判断库存和一人一单。
2. 成功后发送 Kafka 消息。
3. 消费者异步创建订单。

## 笔记接口

### 热门笔记

```http
GET /blog/hot?current=1
```

### 查询笔记详情

```http
GET /blog/{id}
```

### 点赞笔记

```http
PUT /blog/like/{id}
```

### 查询点赞排行榜

```http
GET /blog/likes/{id}
```

## 关注接口

### 关注或取消关注

```http
PUT /follow/{id}/{isFollow}
```

### 查询共同关注

```http
GET /follow/common/{id}
```

### 查询关注 Feed

```http
GET /blog/of/follow?lastId=0&offset=0
```
